Economische aspecten van technologie

Geschreven door Mirja Vink en Eline Pastoor

Voor de minor hebben Eline en ik samengewerkt aan het uploaden van een API. Toen wij dit aan het doen waren, kwamen we erachter dat veel bedrijven API’s gebruiken. Wij zijn een kort onderzoek gaan doen naar de rol die API’s hebben in de huidige samenleving.

De definitie van een API volgens Schoemaker (2019) is: “ Een Application programming Interface (API) is een software-interface die het mogelijk maakt dat twee applicaties met elkaar kunnen communiceren.”

Een groeiend aantal bedrijven maakt gebruik van API’s in hun bedrijfsvoering. Een effectieve API kan bestaande en potentiële klanten nieuwe redenen geven om met een bedrijf te communiceren en hun ervaringen met anderen te delen. API’s kunnen naast de interactie met klanten ook een waardevolle toevoeging zijn aan de interne bedrijfsvoering van een organisatie. Als bijvoorbeeld een software provider een API ontwikkelt, waarbij alle huidige storingen van internet bedrijven op hun website weergegeven worden, zou dit voor klanten een nieuwe reden zijn om met regelmaat op hun site te kijken. (Mulesoft, z.d.)

Een organisatie zou bijvoorbeeld API’s kunnen ontwikkelen voor het gebruik door haar eigen medewerkers. Voorbeelden hiervan zijn:

  • Een API om het verkoopteam nauwkeurige offertes te geven, via mobiele apps.
  • Een API die de klantgegevens makkelijker toegankelijk maakt voor medewerkers van het bedrijf, zodat het hen minder handelingen kost om bepaalde onderdelen af te werken.

Data in de business

Data wordt verzameld tijdens de activiteiten van een bedrijf. Ze kunnen veel meten en registreren, zowel intern als extern. De gegevens zelf zijn niet altijd erg informatief, maar ze vormen wel de basis voor alle rapportage en zijn daarom erg cruciaal in het bedrijfsleven. Een voorbeeld van data die gebruikt wordt in (bijna) elk bedrijf, zijn de klantgegevens. Dit zijn de statistieken die betrekking hebben op klantinteractie. Hieronder kunnen het aantal banen vallen, het ontvangen inkomen of de gemaakte kosten. Bij een bedrijf zijn er bijvoorbeeld gegevens bekend van de debiteuren en crediteuren, welke veel informatie verschaffen over de klanten. Naast op het financiële gebied is data ook belangrijk voor onder andere de marketingafdeling, waar men bezig is met marktsegmentatie (Sirianni, z.d.)

Data kan ervoor zorgen dat processen in een bedrijf worden gestroomlijnd. Tevens kan het zorgen voor kostenbesparing, omdat er bijvoorbeeld niet wordt geadverteerd bij groepen die toch geen interesse hebben. Data kan een grote invloed hebben op de bedrijfsresultaten. De rol van de manager is hierbij belangrijk, aangezien hij de bedrijfsstrategie moet aanpassen aan de hand van deze gegevens (Leonard, 2018).
